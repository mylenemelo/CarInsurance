# Car Insurance MVC App

ASP.NET MVC application that calculates insurance quotes based on user input and stores data in a database.

## Features

- Automatic insurance quote calculation
- Age, car year, make/model, tickets, DUI, and coverage type affect price
- Admin page displays all insured users and their quotes
- Quote is hidden from user input and calculated in controller

## Database

- Uses Entity Framework (LocalDB)
- Stored in App_Data (.mdf / .ldf)
